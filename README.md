## Manual

### check parameters

```
python opendiag --paramaters
```
or

```
python opendiag -PR
```

## get Trouble codes

```
python opendiag --dtc
```
or
```
python opendiag -DT
```

### Help
```
python opendiag --help
```
or
```
python opendiag -H
```
